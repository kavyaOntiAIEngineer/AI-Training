# CNN & OpenCV Assignment

> A comprehensive implementation of fundamental Image Processing, Computer Vision, and Convolutional Neural Network (CNN) concepts using Python, OpenCV, and TensorFlow/Keras.

---

## 📖 Project Overview

This project demonstrates practical implementations of essential Computer Vision and Deep Learning techniques. It covers image preprocessing, pixel-level operations, OpenCV-based image transformations, and the design, training, and evaluation of Convolutional Neural Networks (CNNs).

The objective of this assignment is to build a strong foundation in image analysis and deep learning workflows while applying industry-standard Python libraries.

---

## 🚀 Key Features

### Image Processing
- Image loading using Pillow (PIL)
- RGB to NumPy array conversion
- Image normalization
- Pixel manipulation
- Image reshaping for CNN input
- Image resizing
- Histogram visualization
- Grayscale image processing

### OpenCV Operations
- Image reading and visualization
- Cropping
- Image rotation and flipping
- HSV color space conversion
- Drawing geometric shapes
- Gaussian, Median, and Bilateral filtering
- Binary thresholding
- Edge detection using Canny
- Contour detection
- Complete OpenCV preprocessing pipeline

### Deep Learning (CNN)
- CNN model architecture using TensorFlow/Keras
- Convolution and MaxPooling layers
- Batch Normalization
- Dropout Regularization
- Model Compilation
- EarlyStopping Callback
- Model Saving & Loading
- Performance Evaluation
- Confusion Matrix & Classification Report
- Single Image Prediction
