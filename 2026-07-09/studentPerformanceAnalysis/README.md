# Student Performance Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Students Performance dataset from Kaggle. The main objective is to analyze factors affecting student scores and understand relationships between different features.

## 📂 Dataset
The dataset contains information about students including:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## 🔹 Data Preprocessing

Performed preprocessing steps:

- Loaded CSV dataset
- Checked dataset information
- Handled missing values
- Checked duplicate records
- Renamed columns for better readability
- Created average score feature

## 📊 Exploratory Data Analysis

### 1. Univariate Analysis

Analyzed individual features:

- Gender distribution
- Math score distribution
- Reading score distribution
- Parental education distribution

Visualizations used:

- Count Plot
- Histogram
- Distribution Plot

---

### 2. Bivariate Analysis

Analyzed relationships between two variables:

Examples:

- Gender vs Average Score
- Test Preparation vs Performance
- Lunch Type vs Student Score
- Math Score vs Reading Score

Visualizations used:

- Bar Plot
- Box Plot
- Scatter Plot

---

### 3. Multivariate Analysis

Analyzed multiple features together:

- Correlation between scores
- Impact of gender and test preparation
- Effect of parental education on performance

Visualizations used:

- Heatmap
- Pairplot

## 📈 Key Insights

- Students completing test preparation performed better.
- Reading and writing scores showed strong correlation.
- Lunch type affected student performance.
- Different factors influence academic scores.

## Conclusion

EDA helped identify important factors affecting student performance and understand relationships between academic features.
