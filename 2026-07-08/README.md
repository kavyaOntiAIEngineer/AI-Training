# API and JSON Handling using Python

## 📌 Overview
This repository contains solutions for an API and JSON assignment using Python.  
It demonstrates REST API communication, JSON handling, HTTP methods, and processing API responses.

## 🛠️ Technologies Used
- Python
- Requests Library
- Pandas
- JSON
- REST API

## 📚 Concepts Covered
- API & Client-Server Architecture
- HTTP Methods (GET, POST, PUT, PATCH, DELETE)
- Status Codes
- JSON Data Handling
- Query Parameters
- Timeout & Exception Handling
- Converting JSON data into Pandas DataFrame

## 🌐 API Used
JSONPlaceholder REST API

https://jsonplaceholder.typicode.com

## 💻 Implementations
- Fetch API data using GET request
- Send JSON data using POST request
- Filter data using query parameters
- Handle request errors using try-except
- Convert API response into DataFrame

## 🚀 Learning Outcomes
- Working with REST APIs
- Handling JSON responses
- Sending API requests using Python
- Managing API errors
