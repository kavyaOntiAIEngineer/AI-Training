 # Statistics & Probability Assignment

## Overview
This project demonstrates the application of Statistics and Probability concepts using Python. The analysis is performed on an employee dataset to understand workforce trends, salary distribution, employee performance, and business insights through statistical methods and data visualization.

## Dataset
- **File:** `employee_ai_statistics_dataset.csv`
- **Records:** 20 Employees
- **Features:** 19 Columns

The dataset contains information such as:
- Employee ID
- Department
- Gender
- Age
- Experience
- Salary
- Projects Completed
- Training Hours
- Monthly Sales
- Customer Rating
- Website Visits
- Monthly Watch Time
- Fraud Score
- Performance Rating
- Work Mode
- Employee Sentiment
- Company Status
- Email Type
- Conversion Status

## Objectives
- Load and explore the dataset
- Perform descriptive statistical analysis
- Visualize data using charts
- Calculate probabilities
- Perform sampling techniques
- Compute confidence intervals
- Conduct hypothesis testing
- Perform correlation analysis
- Build a simple Linear Regression model
- Apply Bootstrap Sampling
- Generate business insights
